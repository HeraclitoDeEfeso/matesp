Método que lee un comando del flujo de datos. 

Este método devuelve un árbol sintáctico abstracto que se corresponde con un comando leído del flujo de datos. Precondición\+: el flujo de datos debe estar abierto, debe existir memoria suficiente para crear el árbol sintáctico. Poscondición\+: se leerá del flujo de datos, se solicitará memoria para el árbol sintáctico.
