Método que lee los diversos símbolos de un flujo de datos. 

Este método devuelve en un vector de referencias los símbolos leídos de un flujo de datos. Precondición\+: el flujo de datos debe estar abierto, debe existir memoria suficiente para leer dos líneas del flujo y el vector resultado. Poscondición\+: se leerá del flujo de datos, se solicitará memoria para leer dos líneas del flujo de datos, podría solicitarse memoria para el vector resultado. La última posición del vector es nula.
